## [0.2.0] - 2026-07-01

### Added

- Four-phase training pipeline covering Wake, Dream, Nightmare, and Compress phases
- Command-line interface with `train`, `evaluate`, `benchmark`, and `distort` commands
- FastAPI server with `/distort`, `/evaluate`, `/pipeline`, and `/suggest` endpoints
- Next.js cyberpunk dashboard with 20+ interactive monitoring and control panels
- Multi-model ensemble benchmarking support
- Distributed multi-GPU execution support
- Distortion registry with configurable distortion pipelines
- Comprehensive evaluation and benchmarking framework
- 660+ automated unit tests covering training phases, API endpoints, CLI, and distortion modules
